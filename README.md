## ❗技术栈

- ✅ Web框架：FastAPI
- ✅ Agent框架: LangChain + Llamaindex
- ✅ 向量存储: Milvus
- ✅ DB存储: PostgreSQL
- ✅ 大语言模型: DeepSeekR1-8B (推理) + Qwen-7B-Struct(问答)
- ✅ Embedding模型: BAAI/bge-m3
- ✅ 缓存: Redis
- ✅ 推理引擎: vLLM + Ollama
- ✅ 推理服务: NVIDIA Triton Inference Server
- ✅ 服务部署: Docker
- ✅ 前端框架：React + Ant Design + Gradio
- ✅ 文件存储: Minio
- ✅ 图数据库: Neo4j
